## 🚀 Featured Projects

### 🎸 [Elan Audio Lab](https://elanaudiolab.com)

A full-stack e-commerce platform for custom tube amplifier building. Customers browse, customize components, and order handcrafted amplifiers - all from a single app.

**Stack:** Flutter/Dart • Supabase (PostgreSQL) • Square & PayPal Payments • Cloudflare CDN • GitHub Actions CI/CD

**Platforms:** [Web](https://elanaudiolab.com) • iOS • Android (Google Play)

**Highlights:**
- Real-time product customization with shareable configurations
- Secure checkout (Square + PayPal native SDKs)
- Admin portal with order management, warranty tracking, and analytics
- Automated email system (14 branded templates)
- A-grade security (RLS policies, rate limiting, PII protection)
